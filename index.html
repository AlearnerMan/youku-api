Youku Open API

本文内容来自http://web.archive.org/web/20120814163447/http://dev.youku.com/

简介：

优酷Open API 帮助您在自己的网页、软件或者设备上使用优酷视频平台主要功能。

基本功能：

[上传]

您可以直接上传视频或者通过应用程序（桌面应用或服务器端应用）上传视频到优酷平台，它就有机会被上千万的优酷用户看到。

您只需将视频上传到优酷，那么剩下的工作：存储、转码、播放都将由优酷出具资源向您服务，此外还可以很方便的引用优酷服务器保存的视频截图和Metadata(视频信息)数据。

[聚合]

您可以通过它取得用户相关数据，如：我的视频，专辑，订阅等。

[播放]

您可以构建各种平台上的程序来嵌入优酷视频。

您可以自定义您的"合作播放器"的外观，包括按钮，控制元素和主视觉等，您的企业Logo也可以嵌入其中。


其他特性:

取得XML或JSON格式的聚合数据。

利用JavaScript客户端库直接嵌入优酷视频，而不用使用服务器端代理。


注意：

优酷Open API现阶段只对合作方开放，不对个人开发者开放，您需要申请合作方ID（partnerID / pid）才能使用。
