专辑详细信息

1 简要描述

根据专辑ID来得到专辑详细信息。

GET http://api.youku.com/api_ptplaylistinfo

2 参数

参数 定义 备注
pid YOUKU合作ID
id 专辑ID
rt 返回数据类型 2.XML 3.JSON

3 返回结果

成功: <item status="success">

失败: <item status="fail">

<total />是符合结果的总数

item子节点的定义如下

节点 定义 备注
playlink 专辑播放URL
playswf 专辑播放器UR
name 专辑名称
imagelink 专辑截图
videonum 视频数量
duration 专辑总时长(秒)
user 专辑创建者
uid 专辑创建者ID
description 专辑简介
pv 专辑总播放数
pubdate 发布时间
