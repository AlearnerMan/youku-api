视频详细信息

1 简要描述

根据视频ID得到视频的详细信息。

GET http://api.youku.com/api_ptvideoinfo

2 参数

参数 定义 备注
pid YOUKU合作ID
id 视频ID 兼容优酷播放URL
rt 返回数据类型 2.XML 3.JSON

3 返回结果

成功: <item status="success">

失败: <item status="fail">

item下的节点定义如下：

节点 定义 备注
playlink 视频播放UR
playswf 对外合作播放器
shareswf 站外分享播放器
title 视频标题
imagelink 视频截图
comment 视频描述
duration 时长
pubdate 发布时间
user 视频上传者
uid 视频上传用户ID
pv 视频播放数
totalcomment 视频评论数
cats 视频分类
tags 视频标签 多个组成
fav 视频收藏数
up 视频顶数
down 视频踩数
reference 视频引用数
state 视频状态 0.转码中 1.转码失败 2.待审核 3.正常 4.被屏蔽
publicType 视频权限 0.公开 1.仅好友观看
quality 视频质量
