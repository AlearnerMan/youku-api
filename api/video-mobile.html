移动设备专用播放器

1. 引入JSLoader

请将[PID]替换为优酷分配给您的PartnerID.

2. 在需要嵌入的div 中嵌入JSLoader 播放器

youku.loadPlayer(adapter,videoid,divid,preview,w,h,appid);

3. 参数说明

adapter:适配类型 可以是下面的参数

ios_m3u8 对应ios 设备

android_m3u8 对应Android4.x 设备

flash 对应安装flash 的Android2.x 设备

html5_mp4 使用3gphd

videoid:视频的videoid

divid:要嵌入的div id

preview:视频缩略图,没有缩略图url填空字符串

w:整型值播放器的宽度(像素)

h:整型值 播放器的高度(像素)
