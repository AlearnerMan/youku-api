热点视频

1 简要描述

得到优酷热点视频。

GET http://api.youku.com/api_getHotVideos

2 参数

参数 定义 备注
pid YOUKU合作ID
category 频道名称 如: 资讯
pn 页数 default: 1
pl 页长度 default: 100
ob 排序 default: total_vv
createtime: 最新发布
total_vv: 最多播放
total_up: 最热话题
total_fav: 最多收藏
total_comment: 用户推荐
format 视频格式 flv,flvhd,hd,3gp,3gphd
seconds 视频时长 可按秒数来指定一个范围
如: 100-200
period 发布时间 default: today
today: 今日
week: 本周
month: 本月
history: 历史
all: 全部
total_vv 视频播放量 可按播放数来指定一个范围
如: 10000-20000

3 返回结果

JSON
