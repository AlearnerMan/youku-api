首页推荐视频

1 简要描述

得到优酷首页推荐的视频。

GET http://api.youku.com/api_getIndexRecVideos

2 参数

参数 定义 备注
pid YOUKU合作ID
recommend_type 推荐类型 all.全部
channel.频道推荐视频
category 频道名称 recommend_type=channel时有效
可选值：
今日头条
最佳原创
娱乐资讯
电视资讯
电影资讯
科技财经
汽车
时尚旅游
生活母婴
体育音乐
游戏搞笑
会员推荐
pl 页长度 default: 100
format 视频格式 flv,flvhd,hd,3gp,3gphd
seconds 视频时长 可按秒数来指定一个范围
如: 100-200

3 返回结果

JSON
