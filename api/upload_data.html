纯数据方式上传接口

1 简要描述

以纯数据的方式上传视频到优酷

POST http://gupload.youku.com/upload/uploadPackage

2 参数

参数 定义 备注
partnerid YOUKU合作ID
title 视频标题 标题最多能填写50个汉字
标题不可以只用数字表示
tags 视频标签 单个标签2-6个汉字
标签不能为空
标签个数不能超过10个
单个标签最少2个字母，最多12个字母
单个标签最少2个汉字，最多6个汉字
catIds 视频分类 详细分类见分类聚合
只能填写一个分类
sourceType 版权所有 0. 转载 1. 原创
publicType 视频权限 0. 公开
memo 视频描述
folderIds 上传时加入专辑 多个专辑ID用逗号分隔
username 上传的用户名
password 上传的用户密码 用户密码如果为32位,按md5来验证,否则按照明码验证
Filedata 视频数据

3 返回结果

结果 定义 备注
>0 视频ID 上传成功
0 系统错误
-1 文件过大 限制200M
-2 文件过大 超出了MAX_FILE_SIZE
-3 上传文件不全 服务器只接收到部分数据
-4 上传文件为空 服务器未收到文件数据
-6 临时目录读取错误
-7 写文件错误 服务器保存文件失败
-8 验证失败 用户名密码验证失败
-9 文件信息错误 无Filedata数据
-10 文件信息错误 Filedata大小为0
