视频简短JSON格式定义

1 简要描述

1.1 什么是JSON

JSON(JavaScript Object Notation) 是一种轻量级的数据交换格式。易于人阅读和编写。同时也易于机器解析和生成。它基于JavaScript Programming Language, Standard ECMA-262 3rd Edition - December 1999的一个子集。 JSON采用完全独立于语言的文本格式，但是也使用了类似于C语言家族的习惯（包括C, C++, C#, Java, JavaScript, Perl, Python等）。这些特性使JSON成为理想的数据交换语言。

2 变量定义

2.1 total

符合条件的总条目数

2.2 results

results是一个数组，数组中的每个变量定义如下:

变量 定义 备注
videoid 视频ID
title 视频标题
desc 视频描述
thumburl 截图地址
username 视频上传者
seconds 视频时长 秒
total_comment 视频评论数
total_fav 视频收藏数
total_up 视频顶数
total_down 视频踩数
total_vv 视频播放数
createtime 视频上传时间
