视频RSS格式规范

1 简要描述

1.1 什么是RSS

RSS is a Web content syndication format.

RSS是一种网页内容联合格式。

It's name is an acronym for Really Simple Syndication.

它的名字是Really Simple Syndication的缩写。

RSS is a dialect of XML. All RSS files must conform to the XML 1.0 specification, as published on the World Wide Web Consortium (W3C) website.

RSS是XML的一种。所有的RSS文档都遵循XML 1.0规范，该规范发布在W3C网站上。

2.1 total节点

2.2 item节点

结果由多个item节点组成，item子节点定义如下：

节点 定义 备注
title 视频标题
link 视频播放URL
description 视频截图和描述 <a><img src="截图" /></a> <p>描述</p>
itunes:duration 视频时长
itunes:keywords 视频标签
author 视频上传者
comments 视频评论URL
pubDate 视频发布时间(GMT) Sun, 15 Nov 2009 02:19:31 +0800
guid 视频唯一ID
enclosure 视频播放器
pv 视频播放次数
state 视频状态 0.转码中 1.转码失败 2.待审核 3.正常 4.被屏蔽
publicType 视频权限 0.公开 1.仅好友观看
