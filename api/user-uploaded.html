获取用户上传视频

1 简要描述

根据用户名来聚合视频。

GET http://api.youku.com/api_ptvideo/st_5

2 参数

节点 定义 备注
pid YOUKU合作ID
sv 用户名或Email URL编码
rt 返回数据类型 2.XML 3.JSON
pg 页码 >=1
pz 每页显示数量 1-100
ob 排序类型 1.上传时间 2.播放数 3.评论数量 4.被收藏次数
owner 指定我的视频 my: 我上传的视频(包括未审核和屏蔽的视频)
