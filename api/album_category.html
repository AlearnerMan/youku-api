按分类聚合专辑

1 简要描述

根据分类ID得到视频专辑。

GET http://api.youku.com/api_ptplaylist

2 参数

参数 定义 备注
pid YOUKU合作ID
c 分类ID 0.全部 123.热点 124.原创 128.电影...
详细见专辑页左侧
rt 返回数据类型 2.XML 3.JSON
pg 页码 >=1
pz 每页显示数量 1-100
ob 排序类型 1.最新更新 2.最多播放 4.历史推荐
pd 时间段 1.本日 2.本周 3.本月 4.历史

3 返回结果

成功: <playlist status="success">

失败: <playlist status="fail">

<total />是符合结果的总数

专辑数据是由多个item节点组成,每个item的定义如下:

节点 定义 备注
playlink 专辑播放URL
guid 专辑唯一ID
imagelink 专辑截图
name 专辑名称
intro 专辑简介
user 专辑创建者
uid 专辑创建者ID
videonum 视频数量
totalpv 专辑总播放数
pubdate 发布时间
totalseconds 专辑总时长
video 专辑中前三个视频
